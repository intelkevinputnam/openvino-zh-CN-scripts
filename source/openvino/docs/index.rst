      <p>
		OpenVINO™ 是一个用于优化和部署人工智能（AI）推理的开源工具平台。
		<ul>
            <li>提高计算机视觉、自动语音识别、自然语言处理和其他常见任务的深度学习性能。</li>
            <li>使用经过 TensorFlow、PyTorch 等流行框架训练的模型。</li>
            <li>减少资源需求，并在从边缘到云的一系列英特尔®平台上高效部署。</li>
         </ul>
     </p>
